# hack-gate-entry
1) Download IOT Remote onto phone.
2) Upgrade and include audio, or, magnetic sensor data on dashboard.
3) Place near a gate entry or keypad. Press each button and record all buttons and their natural corresponding magnetic signature from wear and small mechanical differences using screen recording software. Notice precise Y Axis levels.
4) Wait for entries by users. Take signatures, the peaks and troughs and match to recorded signal and corresponding buttons.

Note: Buttons with more wear should work better. Also, maybe applying a coat of various materials, such as very tiny magnetic spheres or even some sort of solder, etc can help to differentiate buttons.
